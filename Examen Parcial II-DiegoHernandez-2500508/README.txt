Este es un programa de consola para gestionar libros de una librería. Dentro del programa se podra realizar gestiones para nuestra libreria
personalizada, en este caso se puede agregar, buscar, eliminar, editar y ver los libros que nosotros querramos.

¿Cómo usarlo?
1. Abre la carpeta del proyecto.
2. Haz doble clic en el archivo 'iniciar.bat' para que se instale todo y empiece.
